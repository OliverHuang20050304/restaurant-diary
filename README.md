# 我的餐廳日記 🍽️

一個優雅、簡潔的餐廳記錄網站，幫助你記錄每一餐的美好時光。

## ✨ 功能特色

### 核心功能
- 📝 **餐廳記錄**：輕鬆新增、編輯、刪除餐廳資訊
- 🏷️ **分類系統**：支援中式、西式、日式、韓式、甜點、咖啡、其他等分類
- 🗺️ **Google Maps 整合**：一鍵開啟 Google Maps 查看餐廳位置
- ⭐ **評分系統**：1-5 星評分，記錄用餐體驗

### 進階功能
- 🔍 **搜尋功能**：快速搜尋餐廳名稱、地址或評論
- 🎯 **分類篩選**：依分類快速篩選餐廳
- ❤️ **收藏功能**：標記最愛的餐廳
- 📅 **造訪日期**：記錄用餐日期
- 💰 **價格標記**：標記價格範圍（$、$$、$$$、$$$$）
- 📝 **評論筆記**：記錄用餐心得、推薦菜色
- 📊 **統計資訊**：總餐廳數、分類數、平均評分、收藏數
- 📸 **照片支援**：可添加餐廳照片（網址）
- 💾 **資料持久化**：使用 localStorage 自動儲存資料

## 🎨 設計特色

- **優雅風格**：採用大地色系，簡約優雅的設計
- **響應式設計**：完美支援手機、平板、桌面裝置
- **流暢動畫**：平滑的過渡效果和互動體驗
- **直覺操作**：簡單易用的介面設計

## 🚀 使用方式

1. 直接在瀏覽器開啟 `restaurant_diary.html`
2. 點擊「+ 新增餐廳」開始記錄
3. 填寫餐廳資訊並儲存
4. 使用搜尋和篩選功能快速找到餐廳
5. 點擊「地圖」按鈕在 Google Maps 中查看位置

## 📋 資料儲存

所有資料都儲存在瀏覽器的 localStorage 中，無需後端伺服器。資料會自動保存，即使關閉瀏覽器也不會遺失。

## 🔧 技術規格

- **純前端**：HTML + CSS + JavaScript
- **無依賴**：不需要任何外部框架或庫
- **本地儲存**：使用 localStorage API
- **響應式設計**：CSS Grid 和 Flexbox

## 📝 注意事項

- 照片功能目前支援圖片網址，如需本地上傳功能需要後端支援
- Google Maps 連結會在新分頁開啟，無需 API Key
- 建議定期備份資料（可匯出 JSON 格式）
